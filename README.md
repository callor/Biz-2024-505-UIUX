# VS Code 설치

- 검색창에 `vsCode` 를 입력하여 `visual Code Studio` 를 찾아 다운로드 및 설치하기

## vsCode 실행하기

- 작업폴더에서 마우스 우클릭하여 vsCode 실행하기
